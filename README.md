# Integrative Social Scaffolding

Interactive visualization of national integrity across Western Christian-heritage nations.

## Framework

This project visualizes country data through the lens of **10 Social Systems**:

1. **Family / Kinship** — Attachment and affect-regulation
2. **Religion / Shared Moral Order** — Superego and transpersonal attachment
3. **Community / Local Institutions** — Affiliation and social bonding
4. **Education / Civic Formation** — Cognitive-moral integration
5. **Economy / Property Order** — Agency and self-efficacy
6. **Law / Justice** — Moral-regulatory and impulse-control
7. **Cultural Canon / Aesthetic Continuity** — Symbolic-integration and narrative-self
8. **Nationhood / Political Identity** — Collective-identity and macro-attachment
9. **Gender / Role Complementarity** — Sexual-polarity and individuation
10. **Speech / Inquiry** — Epistemic-integrity and reality-testing

## Data Sources

- World Values Survey (WVS)
- European Values Survey (EVS)

## Countries

Western Christian-heritage nations: EU, North America, Australia, New Zealand
